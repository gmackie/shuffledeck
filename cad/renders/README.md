# Card Shuffler — Render Assets

STEP files for marketing renders, generated by `render_setup.py`.

## Files

| File | Description |
|------|-------------|
| `assembly_full.step` | Complete assembly — all parts positioned in their correct locations |
| `assembly_exploded.step` | Exploded view — parts offset along Z axis for visual clarity |
| `feeder_detail.step` | Feeder module with Candidate A singulation mechanism |
| `bin_bank_detail.step` | 8-bin bank standalone |

## Generating

```bash
cd cad
python render_setup.py
```

Requires `build123d` (`pip install build123d`).

## Rendering Options

### FreeCAD (Raytracing)

1. Open FreeCAD >= 0.21
2. File > Import > select the `.step` file
3. Switch to the Render workbench (install via Addon Manager)
4. Create a render project (Cycles, LuxCore, or Appleseed backend)
5. Assign materials and set up 3-point lighting
6. Render at 2048x2048 or higher

### Blender (STEP Import)

1. Install [blender_step_importer](https://github.com/30350n/blender_step_importer) or CAD Sketcher addon
2. File > Import > STEP > select the file
3. Assembly imports as separate mesh objects — assign materials in Shader Editor
4. Use an HDRI environment for realistic lighting
5. Render with Cycles at 2048x2048+

**Suggested material palette:**
- Chassis: matte light aluminum
- Bin bank: semi-gloss dark gray
- Feeder / selector: accent blue or orange
- Output tray: same as chassis

## Recommended Camera Angles

### 3/4 Hero Shot (primary marketing image)
- **Camera:** (1200, -800, 600) mm
- **Look-at:** center of bin bank, approximately (400, 50, 30)
- Shows full machine: feeder, bin bank, and output tray in one frame

### Top-Down Overview
- **Camera:** (400, 50, 1000) mm
- **Look-at:** (400, 50, 0)
- Orthographic or long-focal-length perspective
- Shows the card flow path from feeder to output

### Feeder Detail (close-up)
- **Camera:** (80, -120, 100) mm
- **Look-at:** (50, 0, 30)
- Tight shot on singulation mechanism and feed rollers

### Bin Bank Detail
- **Camera:** (400, -200, 200) mm
- **Look-at:** center of bin bank
- Shows 8-bin array with funnel entries visible

### Exploded View (technical illustration)
- Use `assembly_exploded.step`
- **Camera:** (1200, -1000, 800) mm
- **Look-at:** center of assembly
- Shows layered construction: chassis, modules, selector on top
